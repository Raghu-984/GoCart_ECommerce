# 🛒 GoCart – E-Commerce Web App

## 📌 About the Project

GoCart is a simple e-commerce web application built using HTML, CSS, and JavaScript.
It allows users to browse products, view details, add items to cart, and manage their shopping experience.

This project focuses on frontend logic, local storage handling, and dynamic UI updates.

---

## 🚀 Features

* 🛍️ Browse products (fetched from API)
* 🔍 Search products
* 📄 View product details
* 🛒 Add to cart / Remove from cart
* 💾 Cart persistence using localStorage
* 🔐 User Registration & Login system
* 👤 Session handling (login/logout)
* 📱 Responsive UI

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* LocalStorage
* DummyJSON API
